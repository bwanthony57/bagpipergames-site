GOLFER'S PAIR-A-DICE  -  BRAND KIT
Created for BAGPIPER, LLC  -  2026-06-30

WHAT'S INSIDE
  00_brand_sheet/   One-page brand guide (SVG, PNG, PDF).
  01_wordmark/      Primary stacked wordmark "Pair-A-Dice".
  02_lockup/        Horizontal lockup "Golfer's Pair-A-Dice".
  03_crest/         Clubhouse crest emblem.
  04_ballmark/      Compact roundel for golf balls & small uses.
  05_app_icon/      App / store icon (your art) + favicon.ico.
  06_social/        1200x630 social share image.

Every mark ships as scalable SVG plus PNG exports. All SVG text is
converted to outlines, so no fonts need to be installed to use them.

VARIANTS (per mark)
  *_cream / *_green    Full color on the brand cream or green background.
  *_transparent        Full color on a transparent background.
  *_1color_ink         Single-ink (Clubhouse Green) for one-color print,
                       engraving, embossing, stamping.
  *_1color_white       Reversed (white) for dark backgrounds. PNGs are
                       transparent - they look blank on white; place on dark.

WHICH MARK TO USE
  Wordmark   Hero / landing title, wide merch, headers.
  Lockup     Website nav, email signatures, wide-and-short spaces.
  Crest      Premium merch - head covers, towels, tournament gifts.
  Ball mark  Golf-ball side stamp and other small, one-color uses.
  App icon   Apple App Store / Google Play / favicon (square art provided;
             stores apply their own corner rounding).

TYPOGRAPHY
  Baloo 2    - display: wordmark, lockup, crest. (Google Fonts, OFL)
  Fredoka    - labels, eyebrows, UI text.        (Google Fonts, OFL)

CLEAR SPACE & MINIMUM SIZE
  Wordmark / lockup: padding >= cap-height of "Pair-A-Dice".
  Crest / ball mark: clear space >= width of one die.
  Min size - wordmark 120px/1.2in; lockup 200px/2in;
             crest & ball mark 32px screen / 0.5in print
             (use the 1-color version below ~24px).

DON'TS
  Don't recolor, stretch, rotate, add shadows/effects, or place full-color
  marks on busy or low-contrast backgrounds. Use 1-color or reversed instead.
